# Module 10 — PDF Report Generator

## Objective

Generate a complete, professional PDF operation report compiling all project data: vessel and crane configuration, deck layout, crane capacity verification, DNV splash zone analysis results, sea state limits, weather window analysis, and 2D/3D visualization captures.

## User Stories

1. As an engineer, I want to generate a PDF report summarizing the entire operation analysis.
2. As an engineer, I want the report to include 2D deck layout and 3D viewer screenshots.
3. As an engineer, I want the report to include all calculation inputs and results for each equipment item.
4. As an engineer, I want to preview the report before downloading.
5. As an engineer, I want to download the report as a PDF file.

## Route

```
/#/projects/:id/report     → PDF Report Preview & Download
```

## PDF Generation Pipeline

### Step 1: Collect Data
- Load all project data from Zustand stores (already in memory)

### Step 2: Capture Images
1. Navigate to 2D deck layout → capture Konva canvas via `stage.toDataURL()`
2. Navigate to 3D viewer → capture Three.js via `renderer.domElement.toDataURL()`
3. Render sea state tables as HTML → capture via `html2canvas`
4. Render scatter diagram overlay as HTML → capture via `html2canvas`
5. Render crane curve chart via Recharts → capture via `html2canvas`

### Step 3: Assemble PDF
Using jsPDF:
1. Set page size (A4) and margins
2. Add cover page
3. Add table of contents (placeholder — page numbers calculated after assembly)
4. Add vessel/crane section
5. Add equipment list
6. Add deck layout image
7. For each equipment: add analysis pages
8. Add weather window pages
9. Add 3D screenshots
10. Update table of contents with final page numbers
11. Add page numbers to all pages

### Step 4: Download
- Offer PDF as download via `doc.save('SubLift_Report_ProjectName_Date.pdf')`
- Also keep in memory for preview

## Preview

Before downloading, show a page-by-page preview in the UI. Options:
- Render the jsPDF output as a blob URL and display in an `<iframe>`
- Or convert each page to an image and show in a scrollable container

## Report Options (Checkboxes)

The user can include or exclude sections:

| Section | Default | Notes |
|---------|---------|-------|
| Vessel & Crane Summary | ✅ On | Always recommended |
| Equipment List | ✅ On | |
| Deck Layout (2D) | ✅ On | Requires canvas capture |
| Crane Capacity Verification | ✅ On | Per equipment item |
| DNV Analysis Results | ✅ On | Per equipment item |
| Sea State Operability Tables | ✅ On | Per equipment item |
| Weather Window Results | ✅ On | Requires scatter diagram |
| 3D View | ✅ On | Requires 3D scene render |

If a section's data is not available (e.g., no scatter diagram entered), the checkbox is grayed out with "(no data)" label.
